1.Gentoo Projectドキュメントの翻訳

GentooJPのメンバーが中心となって翻訳している翻訳ドキュメントです。 成果はすべてGentoo Linux Projectへ還元しています。

Gentoo Linux Projectのドキュメント一覧(日本語)を参照してくださ い。

2.GentooJPドキュメント

GentooJP開発者によって作成/編集/メンテナンスされているドキュメントです。

タイトル 要約
GentooJPについて GentooJPについての説明が記述されています。
GentooJPの歩き方 Gentoo LinuxまたはGentooJPについて良く知らない方向けに、各ドキュメントやサイトの説明が記述されています。
Gentoo Linux tips集 このガイドはGentooJPのWikiで出たtipsを集めて編集したものです。
GentooJP ニュース投稿について GentooJPではニュースを一般から広く募集しています。 このドキュメントではその手順を説明します。
オリジナルドキュメントプロジェクト GentooJPで掲載を行なっているオリジナルドキュメントに関する説明や参加方法が記述されています。
ebuildプロジェクト Gentoo Linuxの核となるパッケージ記述ファイルであるebuildに関するGentooJPのプロジェクトの説明が記述されています。
翻訳プロジェクト Gentoo Linux Projectのドキュメントを翻訳するGentooJP翻訳プロジェクトへの、 参加方法や注意点などが記述されています。
GWN翻訳プロジェクト Gentoo Linux Projectが毎週発行しているGentoo Weekly Newsletter(GWN)を翻訳するGentooJP GWN翻訳プロジェクトへの、 参加方法や注意点などが記述されています。
本家XMLドキュメントのHTML変換ガイド Gentoo Linux ProjectのXMLドキュメントをHTMLに変換する方法について実例を示しながら説明しています。
GentooJP開発者ドキュメント GentooJP開発者用のドキュメントです。ニュースの投稿の仕方や、 CVSの使い方などが記述されています。

3.オリジナルドキュメント

著者の方々の好意によってGentooJPで場所を提供し掲載している記事です。

タイトル 著者 要約
Window Maker 真藤直観 Window Maker は NEXTSTEP(TM)のLook&Feel をもつ Window Manager です。この文書では、GentooLinux でのWindow Maker について説明します。
IRCクライアントの設定 (XChat編) Kojima このガイドでは、XChatを使った、GentooJP・IRCチャンネルへの接続について解説します。対象 は、(Gentoo) Linux 上で、初めてIRCに参加するユーザーです。
IRCクライアントの設定(XChat-2編) Kojima このガイドでは、XChatを使った、GentooJP・IRCチャンネルへの接続について解説します。対象 は、(Gentoo) Linux 上で、初めてIRCに参加するユーザーです。
IRCクライアントの設定 (LimeChat編) Takashi Ota. このガイドでは、LimeChatを使った、GentooJP・IRCチャンネルへの接続について解 説します。対象は、Microsoft Windows 上で、初めてIRCに参加するユーザー です。
Gentoo Linux フロッピーインストールガイド yamakura このガイドでは、CD-ROMドライブが使えないマシンに、フロッピーディスクを使っ て Gentoo Linux をインストールする方法を解説します。
翻訳とオリジナルドキュメント作成環境の構築ガイド Takashi Ota このガイドでは、GentooJPの翻訳・オリジナルドキュメント作成の際に必 要となる環境の構築方法について説明します。
Gentoo Linux フェイルオーバーガイド Takuya Asada このドキュメントでは、Gentoo Linuxが予めインストールされた運用系・待機系の2台のサーバの上に、HeartbeatとDRBDというソフトウェアを用いてフェイルオーバシステムを構築します。
Gentoo Linux 負荷分散ガイド Takuya Asada このドキュメントでは、Gentoo Linuxが予めインストールされたサーバ群（４台）の上に、LVS,Ldirectord,NFS,Heartbeat,DRBD等のソフトウェアを用いて高可用性を持つ負荷分散システムを構築します。

タイトル 著者 翻訳者 要約
UDEV Primer David Babcock 五十嵐正尚 udevをセットアップする手順から、ちょっとしたtipsまでGentooを使って解説されています。 Gentoo udevガイド、Writing udev rulesからもリンクされているudevに関する貴重なドキュメントです。
