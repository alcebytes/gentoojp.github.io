翻訳プロジェクト9月の活動内容

Posted on 2011-10-10 by naota

GentooJP翻訳プロジェクトは、2011年9月の活動内容を公表します。 9月はx86/amd64/PPCハンドブックを中心に5つの文書の差分翻訳を行ないました。 また、査読がおいついていませんが、新たに3名の方に翻訳に参加していただきました。

  • Gentoo Linux/PPC ブロックデバイス入門 (version. 2.00 (2005-03-28) → 10.1 (2010-07-20))
  • Gentoo Linux Portage設定 (version. 9.5 (2010-05-14) → 13 (2011-08-13))
  • Gentoo Linux はじめに (ネットーク設定) (version. 8.0 (2007-05-07) → 9 (2011-08-13))
  • Gentoo Linux オフィシャルツリーからの変更 (version. 1.6 (2007-02-26) → 1.7 (2010-07-13))
  • Gentoo Linux Portageのツール (version. 1.5 (2008-03-01) → 2 (2010-10-04))

GentooJP翻訳プロジェクトは引き続き、特に迅速な更新が望まれる次の文書に対して、翻訳者を募集しています。

  • Gentoo ハンドブック
  • BaselayoutとOpenRCの移行ガイド
  • Gentoo Linux x86クイックインストールガイド
  • X サーバー設定ガイド
  • Gentoo Linux ALSA ガイド

また、以下の文書の査読者も募集しています。

  • Gentoo Linux Kernel Configuration Guide

翻訳に興味があり参加してみたい方は、ぜひGentooJP翻訳プロジェクト をご覧ください。
