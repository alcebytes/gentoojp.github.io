翻訳プロジェクト8月の活動内容

Posted on 2011-09-06 by morota

GentooJP翻訳プロジェクトは、2011年8月の活動内容を公表します。 x86/amd64/PPCハンドブックを中心に16の文書に及ぶ差分翻訳が行われました。

  • Gentoo Linux/AMD64 よくある質問とその回答(FAQ) (rev. 1.37 -> 1.38)
  • Gentooインストールファイルをインストールする (rev. 1.122 -> 1.123)
  • Gentooベースシステムのインストール (rev. 1.112 -> 1.119)
  • カーネル設定(x86/amd64) (rev. 1.35 -> 1.37)
  • システム設定 (rev. 1.101 -> 1.102)
  • 必要なシステムツールをインストールする (rev. 1.48 -> 1.49)
  • ブートローダを設定する(x86/amd64) (rev. 1.17 -> 1.23)
  • Portageについて (rev. 1.74 -> 1.75)
  • USEフラグ (rev. 1.49 -> 1.50)
  • Portageの機能 (rev. 1.36 -> 1.37)
  • initスクリプト (rev. 1.31 -> 1.32)
  • ファイルとディレクトリ (rev. 1.20 -> 1.21)
  • ソフトウェアのブランチを併用する (rev. 1.15 -> 1.18)
  • 特別なハードウェアの構成 (rev. 1.2 -> 1.6)
  • 適切なインストールメディアの選択(PPC) (rev. 1.44 -> 1.66)
  • Gentoo Linux PPC ハンドブック (rev. 1.147 -> 1.159)

GentooJP翻訳プロジェクトは引き続き、特に迅速な更新が望まれる次の文書に対して、翻訳者を募集しています。

  • Gentoo ハンドブック
  • BaselayoutとOpenRCの移行ガイド
  • Gentoo Linux x86クイックインストールガイド
  • X サーバー設定ガイド
  • Gentoo Linux ALSA ガイド

翻訳に興味があり参加してみたい方は、ぜひGentooJP翻訳プロジェクト をご覧ください。
